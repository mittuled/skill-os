## Executive Summary

[2-3 sentences summarizing the GTM activation: target segment, launch date, pipeline generation target, and number of reps activated.
GUIDANCE: Lead with the revenue target and timeline. Do not describe the planning process.]

## GTM Summary (Sales View)

[Sales-relevant elements extracted from the GTM plan.

GUIDANCE:
- Good: Table with rows for target segment, value proposition (one sentence), competitive positioning (primary differentiator), launch date, expected inbound volume (MQLs/week), campaign channels, and sales-critical dependencies
- Bad: "We are launching a new product"
- Format: Structured table with one row per element. Flag any missing elements as blockers.]

## Pipeline Waterfall Model

[Pipeline targets working backward from revenue goal.

GUIDANCE:
- Good: Table showing revenue target, required pipeline (at coverage ratio), pipeline by stage (leads needed, meetings needed, proposals needed), stage conversion rates (actual from CRM data), weekly generation targets for the activation period, and target ACV
- Bad: "We need $1M in pipeline"
- Format: Waterfall table flowing from revenue goal down to weekly activity targets. Include assumptions and data sources for conversion rates.]

## Outreach Sequences

[Persona-specific outreach sequences.

GUIDANCE:
- Good: One sequence per buyer persona with: persona name, channel mix (email/phone/LinkedIn/events), number of touches, cadence (days between touches), and messaging theme per touch aligned to GTM value prop. Include subject line examples and call talk track bullets.
- Bad: "Send emails to prospects"
- Format: One subsection per persona with a touch-by-touch table (Touch #, Day, Channel, Message Theme, CTA)]

## Account Assignment Roster

[Target accounts assigned to reps.

GUIDANCE:
- Good: Table with account name, tier (1/2/3), assigned rep, territory/vertical, current status (net-new/existing), and priority rationale
- Bad: "Reps should work their territories"
- Format: Sortable table. Include summary row with total accounts, accounts per rep, and tier distribution.]

## Recommendations

[Readiness items and gap remediation.

GUIDANCE: Each recommendation should be:
- Specific (not "prepare materials" but "Sales Manager delivers battle card for [competitor] by [date]")
- Actionable (assignable to a person/team)
- Prioritized (P1 = blocks launch / P2 = degrades launch / P3 = nice-to-have)]
