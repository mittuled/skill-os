# GTM Activation (Sales) — Example Input

## Scenario

PeopleMetrics is launching into the mid-market fintech vertical in Q2 2026. Marketing has set a campaign launch date of April 15. The Sales Manager needs to validate that sales is ready to activate with pipeline targets, outreach sequences, and enablement in place before the campaign goes live.

}
