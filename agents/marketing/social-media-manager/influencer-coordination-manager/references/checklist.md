# Checklist: influencer-coordination-manager

## Phase 1: Influencer Selection

- [ ] Programme brief approved by marketing lead (ICP, goals, budget, success metrics)
- [ ] Candidate list built with minimum 10 candidates per tier targeted
- [ ] Each candidate scored on all 5 vetting criteria (see framework.md)
- [ ] Shortlist achieves composite score ≥ 3.5/5 on vetting criteria
- [ ] Audience demographics verified via platform analytics or third-party tool
- [ ] Engagement authenticity checked (no sudden follower spikes, substantive comments)
- [ ] Brand safety audit complete (content history reviewed, no controversial posts)
- [ ] No undisclosed competitor partnerships in last 90 days

## Phase 2: Contract and Onboarding

- [ ] Deliverables defined: post types, quantities, platforms, publish dates
- [ ] Compensation terms agreed and documented
- [ ] Exclusivity window specified (if applicable)
- [ ] Usage rights granted for organic repurposing and paid amplification
- [ ] FTC disclosure requirements included as contractual obligation
- [ ] Takedown rights clause included for brand-unsafe content
- [ ] Signed agreement on file before any deliverable work begins
- [ ] UTM link or promo code assigned and tested

## Phase 3: Campaign Execution

- [ ] Creative brief sent with key message, proof points, guardrails, and schedule
- [ ] Draft review deadline communicated (minimum 72 hours before publish)
- [ ] Draft reviewed for brand safety, factual accuracy, and FTC disclosure
- [ ] Feedback provided within 24 hours of draft receipt
- [ ] Final content approved before publication
- [ ] Publication date confirmed with influencer
- [ ] Content live-checked on publish day: disclosure present, UTM link functional

## Phase 4: Performance Tracking and Renewal

- [ ] Engagement metrics captured 48 hours post-publish
- [ ] UTM conversion data pulled from analytics platform
- [ ] CPE and EMV calculated per influencer
- [ ] Pipeline attribution reviewed with demand gen lead
- [ ] Renewal decision made using decision matrix (framework.md)
- [ ] Performance data added to influencer tracking log
- [ ] Feedback shared with influencer (engagement summary, what resonated)
