# Checklist: company-offsite-producer

## Pre-Planning (12+ Weeks Out)

- [ ] Offsite objectives agreed with leadership (strategic alignment / team building / planning / celebration)
- [ ] Dates locked and confirmed with leadership and HR
- [ ] Headcount confirmed including remote/international attendees
- [ ] Budget envelope approved by finance
- [ ] Offsite brief document drafted and approved

## Venue and Logistics (8–12 Weeks Out)

- [ ] Venue shortlist of 3–5 options created using selection criteria (framework.md)
- [ ] Site visits completed for top 2 venues
- [ ] Venue contract negotiated and signed
- [ ] Accommodation block secured (if applicable)
- [ ] Catering options confirmed with dietary accommodation capability
- [ ] AV capabilities confirmed: screens, mics, recording, hybrid support
- [ ] Travel booking mechanism opened (group booking or platform invite)
- [ ] Ground transportation plan in place (airport transfers, shuttle)

## Agenda and Content (5–8 Weeks Out)

- [ ] Draft agenda created using 60/40 framework (framework.md)
- [ ] Session owners assigned for all structured sessions
- [ ] Team-building format selected and booked (format matched to goal — see framework.md)
- [ ] Agenda reviewed and approved by leadership
- [ ] Speaker briefs sent to all session owners (format, time limit, outputs expected)
- [ ] Evening social activities planned with opt-in structure

## Pre-Event Distribution (2–4 Weeks Out)

- [ ] Pre-read materials compiled and reviewed
- [ ] Pre-reads distributed to all attendees at least 14 days before event
- [ ] Dietary and accessibility requirements collected from all attendees
- [ ] Final logistics sheet sent: venue address, transport, schedule, emergency contacts
- [ ] Room layouts and AV requirements confirmed with venue
- [ ] Run-of-show document drafted with minute-by-minute schedule

## On-Site Execution (Day Before + Day Of)

- [ ] On-site setup complete day before: rooms, AV, signage, materials
- [ ] AV test completed in all session rooms
- [ ] Catering confirmed for all meal service times with dietary accommodations verified
- [ ] Session scribes assigned for all strategic working sessions
- [ ] Emergency contacts and first aid location communicated to all staff
- [ ] Real-time schedule monitoring: enforce start/end times for all sessions

## Post-Event (Within 2 Weeks)

- [ ] Feedback survey deployed within 48 hours of event end
- [ ] Session output documents compiled and distributed within 5 business days
- [ ] Action items with owners published to project management tool
- [ ] Budget actuals vs. plan reconciled
- [ ] Post-offsite report delivered to leadership within 2 weeks
- [ ] Recommendations captured for next offsite planning
