# Framework: company-offsite-producer

This framework defines the planning timeline, venue evaluation criteria, agenda design principles, and logistics requirements for producing a company offsite.

## Planning Timeline

| Weeks Before Event | Milestone | Owner | Key Deliverable |
|-------------------|-----------|-------|----------------|
| 12–16 weeks | Objectives locked | Leadership + Event Manager | Offsite brief: dates, headcount, goals |
| 10–12 weeks | Venue shortlist and site visits | Event Manager | Venue comparison matrix |
| 8–10 weeks | Venue contract signed | Event Manager + Finance | Signed contract, deposit paid |
| 6–8 weeks | Agenda designed and approved | Leadership + Event Manager | Draft agenda with session owners |
| 5–6 weeks | Travel logistics launched | Event Manager + HR/Ops | Travel booking platform open; group flights sourced |
| 4–5 weeks | Session owners briefed | Event Manager | Speaker briefs, format guides, time limits |
| 3–4 weeks | Pre-reads distributed | Event Manager | Attendee pre-read packet |
| 2–3 weeks | AV and room layout confirmed | Event Manager + Venue | AV checklist, room set diagrams |
| 1–2 weeks | Final headcount and dietary/accessibility needs confirmed | Event Manager + HR | Final logistics sheet |
| 3–5 days | Run-of-show finalised | Event Manager | Minute-by-minute schedule |
| Day before | On-site setup and AV check | Event Manager | Setup complete by 5 PM |
| Day of | Event execution | Event Manager | Real-time schedule management |
| 24–48 hours after | Feedback survey deployed | Event Manager | Survey link sent to all attendees |
| 1–2 weeks after | Post-offsite report delivered | Event Manager | Report with decisions, actions, feedback, budget actuals |

## Venue Selection Criteria

Score each venue 1–5 on the following. Require minimum 3.5/5 composite to shortlist.

| Criterion | Weight | 1 (Fail) | 3 (Pass) | 5 (Strong) |
|-----------|--------|----------|----------|------------|
| Accessibility | 20% | > 3 hrs travel for > 30% of attendees | < 2 hrs for most | < 1 hr for all; direct flights available |
| Plenary space capacity | 20% | Fits < 80% of headcount | Fits 100% at theatre-style | Fits 100% with classroom or boardroom options |
| Breakout room availability | 15% | 0–1 breakout rooms | 2–3 breakout rooms of adequate size | 4+ breakout rooms with flexible sizing |
| On-site accommodation | 15% | No accommodation; distant hotels | Partnered hotels within 15 min | On-site accommodation blocks for full group |
| AV infrastructure | 15% | No built-in AV; full rental required | Basic AV (screen, projector, mics) | Full production capability; in-house AV team |
| Budget fit | 15% | > 20% over budget envelope | Within 10% of budget | ≥ 15% below budget with no quality compromise |

## Agenda Design Principles

| Principle | Description | Application |
|-----------|-------------|-------------|
| 60/40 structure | 60% structured sessions, 40% unstructured time | Leave at least 2–3 unstructured hours daily for hallway conversations |
| Energy management | High-intensity strategic sessions in morning; workshops or social in afternoon | Do not schedule vote-taking or strategic decisions after 3 PM |
| Interaction density | No session longer than 90 minutes without a break or format change | Every 90-minute block should include ≥ 20 minutes of interaction |
| Single-thread mornings | Plenary sessions in the morning when attention is highest | Reserve breakout tracks for afternoons |
| Concrete outputs | Every working session must produce a documented output | Assign a scribe to every strategic session; output emailed same day |
| Optional social | Evening social activities should be opt-in | Provide quiet alternative for introverts; never mandate attendance |

## Agenda Format: 2-Day Offsite Template

| Slot | Day 1 | Day 2 |
|------|-------|-------|
| Morning (9–12) | Welcome + context-setting keynote → Strategic priority workshop | Cross-functional working sessions (2–3 parallel tracks) |
| Lunch (12–1:30) | Informal lunch — unstructured | Informal lunch — unstructured |
| Afternoon (1:30–5) | Breakout workshops by function or project | Synthesis session: decisions, owners, timeline → Wrap + commitments |
| Evening (6–10) | Team dinner (opt-in social activity) | Optional farewell dinner or early departure |

## Team-Building Format Selection

| Goal | Recommended Format | Duration | Group Size |
|------|-------------------|----------|-----------|
| Cross-functional relationship-building | Structured dinner seating (mixed teams) | 2–3 hours | Any |
| Creative problem-solving | Hackathon or design sprint | 2–4 hours | 6–30 |
| Physical energy release | Outdoor activity (hike, cooking class, sports) | 2–3 hours | 10–60 |
| Light fun / laughter | Trivia, escape room, scavenger hunt | 1–2 hours | 10–100 |
| Reflection and values | Facilitated retrospective or values exercise | 1–2 hours | 10–50 |

## Logistics Checklist Categories

- **Travel**: Flight or drive time thresholds; group booking deadlines; visa requirements for international attendees
- **Ground transportation**: Airport transfers; venue shuttles; rideshare guidance
- **Dietary**: Collection of dietary requirements ≥ 3 weeks before; confirmation with venue catering
- **Accessibility**: Mobility, sensory, and religious accommodation requests; venue accessibility audit
- **AV**: Screen size per room; microphone types (lapel, handheld, table); recording setup for hybrid attendees
- **Connectivity**: WiFi capacity for headcount; dedicated event SSID; backup internet line
- **Emergency**: First aid kit location; nearest hospital; emergency contact list; on-site first aider confirmed
