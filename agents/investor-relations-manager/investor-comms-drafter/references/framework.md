# Framework: investor-comms-drafter

Defines the structural framework for drafting investor communications across update types, narrative standards, and engagement mechanics.

## Communication Type Matrix

| Communication Type | Frequency | Trigger | Primary Audience | Format |
|-------------------|-----------|---------|-----------------|--------|
| Monthly investor update | Monthly | Calendar — first week of month | All cap table holders | Email (500-800 words) |
| Material event update | Ad hoc | Defined material events (see below) | All or subset of cap table | Email (200-400 words) |
| Board pack cover memo | Quarterly | Board meeting | Board directors only | Memo (1 page) |
| Fundraise announcement | Per round | Round close | All + media (if public) | Email + press release |
| Ask-specific outreach | As needed | Specific help needed | Targeted investors | Short email (100-200 words) |

### Material Event Triggers (Always Communicate Within 5 Business Days)

- Revenue milestone (ARR >$1M, $5M, $10M, etc.)
- Strategic hire (C-suite, VP-level)
- Strategic partnership or pilot with a significant customer
- Major competitive development
- Cash concern or runway change >2 months
- Regulatory, legal, or compliance matter
- Pivot or significant strategy change

## Monthly Update Structure

### Section Order and Word Limits

```
1. Subject line: [Company] — [Month YYYY] Update
2. Highlights (3 bullet points max, 100 words total)
3. Key Metrics Table (no prose — numbers only)
4. Lowlights / Challenges (1-2 items, 100 words — always include)
5. Asks (3 items max, 50 words each)
6. Outlook (what to watch in the coming month, 50 words)
```

Total word limit: 600 words. Investors read dozens of updates; brevity is respect.

### Key Metrics Table Format

Include exactly these metrics; do not add or remove without explanation:

| Metric | Current | Prior Month | MoM Change |
|--------|---------|------------|-----------|
| ARR | | | |
| MRR Growth (%) | | | |
| NRR (%) | | | |
| Gross Churn (%) | | | |
| Burn Rate (monthly) | | | |
| Runway (months) | | | |
| Headcount | | | |

Add 1-2 company-specific metrics (e.g., pilot count, design partner NPS) that are most relevant to the stage. Cap at 10 metrics total.

### The Lowlights Rule

Every investor update must include at least one item under Lowlights. If everything genuinely went well, report on a risk or uncertainty the team is monitoring. Investors who receive updates with no lowlights learn to distrust them.

Good lowlight: "New ARR came in at $210K vs. $310K target — 3 enterprise deals slipped to Q2 following longer-than-expected legal review cycles. We are adjusting Q2 pipeline coverage to account for similar delays."

Bad lowlight: "No significant issues to report this month."

### Ask Format

Each ask must follow this structure to be actionable:
```
[Type]: [Specific request] — [One sentence of context]
```

Examples:
- **Intro**: Know anyone at Salesforce (VP of Revenue Operations or above)? We have a strategic partnership conversation we want to start.
- **Hiring referral**: Looking for a VP of Marketing with PLG experience, ideally from a Series A/B SaaS company. Anyone in your network?
- **Advice**: Navigating our first enterprise customer compliance questionnaire — have you seen a good process for this at portfolio companies?

## Tone and Voice Standards

### Register

Investor updates are professional correspondence between equals — not a pitch and not a status report to a manager. The tone should be:
- Direct: Lead with the number or fact, not background
- Honest: Surface challenges before the investor notices them independently
- Confident: Setbacks are normal; the update demonstrates thoughtful analysis, not panic
- Brief: Every sentence earns its place

### Prohibited Phrases

These phrases signal marketing over substance — remove on sight:
- "Exciting news"
- "We're thrilled to announce"
- "We're on a great trajectory"
- "The team is working hard"
- "We believe in our mission"

### Numbers Rule

Every claim of progress must have a number attached:
- Not "we grew significantly" → "MRR grew 12% month-over-month"
- Not "strong pipeline" → "pipeline of $2.1M across 14 opportunities"
- Not "good retention" → "NRR of 118%, up from 112% last quarter"
