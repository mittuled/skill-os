# Checklist: data-room-manager

Covers every step required to maintain an active, current data room across the fundraising lifecycle and ongoing investor relations.

## Document Currency Audit (Monthly)

- [ ] Review every section of the data room against the following refresh schedules:
  - Financial statements and KPI dashboard: refresh monthly (within 15 days of month-end)
  - Cap table: refresh within 48 hours of any equity event (new grant, SAFE, exercise, transfer)
  - Board minutes: upload within 2 weeks of each board meeting
  - Legal agreements: review quarterly — flag any contracts that have been amended or executed since the last update
  - Product materials (roadmap, screenshots, demo): refresh at each major product release
  - Org chart and team bios: refresh when any director-level hire or departure occurs
- [ ] Generate a staleness report: list every document in the data room with its current date vs. required refresh date
- [ ] Assign update tasks to the relevant document owner with a 5-business-day deadline
- [ ] Confirm updates are uploaded before granting any new investor access

## Access Management

- [ ] Access tiers are enforced per the data room access policy:
  - Full access: current investors, signed-NDA diligence-stage investors
  - Standard access: warm prospects who have had at least one meeting — excludes financials and legal
  - Request-only: cold prospects — no self-serve access
- [ ] Access is granted within 24 hours of investor confirming diligence interest — never before the first meeting
- [ ] Every access grant is logged with: investor name, firm, date granted, access tier, and who approved
- [ ] Access is revoked immediately when an investor passes or when a transaction closes — do not leave zombie access grants open
- [ ] Watermarking is enabled for all PDF documents — every download includes investor name and access date
- [ ] Download restrictions are set: sensitive documents (detailed customer lists, employee compensation) require request-only download
- [ ] Quarterly: audit all active access grants — remove any with no activity in 60+ days

## Engagement Tracking

- [ ] Review engagement analytics weekly during an active fundraise
- [ ] Track per investor: which sections they viewed, time spent per document, download activity
- [ ] Flag high-engagement signals to CEO: investor who views financials 3+ times or downloads the cap table is a warm signal
- [ ] Flag low-engagement: investor granted access who views nothing within 5 business days — follow up to confirm receipt
- [ ] Use engagement data for meeting preparation: if an investor viewed the financial model 4 times, expect detailed financial questions
- [ ] Never share engagement analytics with investors — this is internal intelligence only

## Pre-Diligence Completeness Check

Run this before granting access to any diligence-stage investor.

### Tier 1 — Must-Have (block investor access if missing)

- [ ] Certificate of Incorporation and any amendments
- [ ] Bylaws (current version)
- [ ] Cap table — fully diluted, including all SAFEs and outstanding options
- [ ] Financial statements — trailing 12 months minimum
- [ ] Financial model with 3-year projections
- [ ] IP assignment agreements for all founders and key employees

### Tier 2 — Should-Have (complete within 5 days of access grant)

- [ ] All board meeting minutes (from incorporation)
- [ ] All existing investor agreements (SAFE notes, side letters, investor rights agreements)
- [ ] Material customer contracts (top 5 by ARR)
- [ ] Key employment agreements (founders, C-suite)
- [ ] Patent applications or granted patents (if applicable)
- [ ] Pitch deck (current version)

### Tier 3 — Nice-to-Have (complete within 2 weeks of access grant)

- [ ] Customer cohort analysis
- [ ] Detailed product roadmap
- [ ] Reference customer list (with contacts)
- [ ] Competitor comparison
- [ ] Technical architecture overview

## Post-Round Data Room Restructuring

After each funding round closes:

- [ ] Archive the active data room as "[Company] — [Round] — Closing Data Room — [Date]"
- [ ] Revoke all investor access from the closed round's diligence process
- [ ] Grant access to the new investor under the ongoing data room with the current terms
- [ ] Update the data room with all closing documents: executed investment agreements, updated cap table, board consents approving the round
- [ ] Begin updating the data room for the next round: update financial model, refresh KPI dashboard, add latest board minutes
