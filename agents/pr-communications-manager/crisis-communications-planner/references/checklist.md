# Checklist: Crisis Communications Planner

Operational checklist covering pre-crisis preparation, active crisis response, and post-crisis review.

## Phase 1: Pre-Crisis Preparation

### Scenario Planning
- [ ] Crisis scenario matrix drafted covering at minimum: data breach, product outage, executive misconduct, regulatory action, negative press, layoffs
- [ ] Each scenario rated by probability (L/M/H) and impact (L/M/H)
- [ ] Top 5 scenarios (highest combined score) have fully developed response playbooks
- [ ] Scenario matrix reviewed with legal, HR, and executive team
- [ ] Scenario matrix reviewed and updated at least every 6 months or after any material company change

### Escalation Protocol
- [ ] Severity tiers defined (Tier 1–4) with clear criteria for each
- [ ] Escalation decision tree documented: who notifies whom, in what sequence, within what timeframe
- [ ] Primary and backup contacts documented for: CEO, CPO, Legal, HR, Communications, Finance
- [ ] All contacts have personal mobile numbers on file (not just corporate email)
- [ ] Response time SLAs defined and acknowledged by all parties

### Spokesperson Readiness
- [ ] Primary spokesperson designated per scenario category
- [ ] Backup spokesperson identified for each primary
- [ ] All designated spokespersons have completed media training within the last 12 months
- [ ] Spokesperson roster reviewed after any leadership changes
- [ ] Clear guidance documented on who can and cannot speak to media externally

### Messaging Templates
- [ ] Holding statements drafted for all top-5 scenarios
- [ ] Channel-specific versions prepared: press statement, social media, customer email, employee memo
- [ ] All templates reviewed and pre-approved by legal
- [ ] Templates stored in a shared location accessible without corporate VPN (crisis may take down internal systems)
- [ ] Template version control in place; outdated versions archived

### Infrastructure
- [ ] Dark site domain registered and template page built
- [ ] Update access configured for PR Manager independent of IT change management
- [ ] Media monitoring tool configured with crisis keyword alerts and escalation thresholds
- [ ] Emergency communications distribution list maintained and tested quarterly
- [ ] Personal contact list for top-10 journalists maintained offline

---

## Phase 2: Active Crisis Response

### First 30 Minutes (Tier 1) / First 2 Hours (Tier 2)
- [ ] Crisis confirmed and severity tier assessed
- [ ] Response team notified per escalation protocol
- [ ] Holding statement selected and customised from template library
- [ ] Legal sign-off obtained on holding statement
- [ ] Holding statement issued on primary channel
- [ ] Internal stakeholder briefing sent
- [ ] Dark site activated if Tier 1 event
- [ ] Monitoring alerts elevated to real-time frequency

### During Crisis
- [ ] Designated update cadence established (e.g., every 2 hours for Tier 1)
- [ ] All external statements logged with timestamp and channel
- [ ] Inbound media enquiries logged and triaged; only designated spokesperson responds
- [ ] Social media monitored; rapid responses posted to factual questions
- [ ] Internal team briefed on what they can and cannot say externally
- [ ] Legal and HR looped in for any statements involving personnel or liability
- [ ] Fact-check mechanism active: no statement issued without source verification

### Communications Decisions
- [ ] Interview requests reviewed and approved/declined by PR Manager
- [ ] Exclusive offer considered for top-tier journalists to control narrative
- [ ] Customer communications drafted and sent within response window
- [ ] Regulatory or investor notification assessed and executed if required

---

## Phase 3: Post-Crisis Review

### Immediate (Within 1 Week)
- [ ] Timeline of events documented from first alert to final statement
- [ ] All external statements archived with timestamps
- [ ] Media coverage collected and sentiment assessed
- [ ] Internal debrief conducted with response team
- [ ] Gaps in escalation protocol identified and assigned for remediation

### 30-Day Review
- [ ] Sentiment recovery tracked against pre-crisis baseline
- [ ] Customer trust survey launched
- [ ] Scenario matrix and response templates updated based on lessons learned
- [ ] Any spokesperson changes actioned
- [ ] Updated playbook distributed to all stakeholders

### Ongoing
- [ ] Tabletop exercise scheduled within 90 days of any Tier 1 or 2 event
- [ ] Annual full-scenario tabletop exercise on calendar
- [ ] Crisis plan version history maintained
