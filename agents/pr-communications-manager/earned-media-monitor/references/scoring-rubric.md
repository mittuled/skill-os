# Scoring Rubric: earned-media-monitor

Evaluates the quality of earned media tracking, share-of-voice analysis, and sentiment assessment accuracy.

## Criteria

| # | Criterion | Weight | Description |
|---|-----------|--------|-------------|
| 1 | Coverage Completeness | 25% | Whether all relevant brand and competitor mentions are captured across outlet tiers |
| 2 | Share of Voice Accuracy | 25% | Whether SOV calculations are weighted by outlet authority and reach, not just raw mention count |
| 3 | Sentiment Classification | 25% | Whether sentiment analysis correctly categorises coverage and identifies dominant narrative themes |
| 4 | Actionability of Flags | 15% | Whether flagged items include clear context and recommended response actions |
| 5 | Reporting Cadence | 10% | Whether reports are delivered on schedule with consistent methodology period over period |
| **Total** | | **100%** | |

## Scale

Each criterion is scored **0-10**:
- **0**: No evidence / completely absent
- **5**: Partially present with significant gaps
- **10**: Fully present, comprehensive, no gaps

**Composite Score** = Σ (criterion score x weight)

## Grade Bands

| Grade | Composite Score | Label | Description | Recommended Action |
|-------|----------------|-------|-------------|-------------------|
| A+ | 9.0 - 10.0 | Comprehensive | Full coverage across tiers with accurate SOV, sentiment, and actionable flags | Distribute to leadership; use for strategic planning |
| A | 8.0 - 8.9 | Strong | Minor outlet gaps but SOV and sentiment analysis are reliable and actionable | Distribute with noted coverage gaps; close gaps next cycle |
| B | 7.0 - 7.9 | Adequate | Most coverage captured; SOV directionally correct; some sentiment misclassifications | Use for directional insights; improve classification rules |
| C | 5.0 - 6.9 | Partial | Significant outlet or competitor gaps; SOV unreliable for budget decisions | Supplement with manual research before using for decisions |
| D | 3.0 - 4.9 | Unreliable | Major coverage blind spots; sentiment analysis shows high misclassification rate | Audit monitoring tool configuration; do not distribute as-is |
| F | 0.0 - 2.9 | No Basis | Monitoring not operational; no coverage data available for the reporting period | Reconfigure monitoring tool; establish baseline keyword set |

## Signal Tables

### Coverage Completeness

| Score | Evidence |
|-------|----------|
| 9-10 | All tier-1, tier-2, and tier-3 outlets monitored; brand name variations and product names captured; 3+ named competitors tracked; zero known coverage missed |
| 7-8 | Tier-1 and tier-2 outlets fully covered; tier-3 has minor gaps; 2-3 competitors tracked; occasional missed mention in niche outlets |
| 5-6 | Tier-1 covered but tier-2 has gaps; only 1-2 competitors tracked; monitoring keywords miss common brand misspellings |
| 3-4 | Only tier-1 outlets monitored; no competitor tracking; significant keyword gaps |
| 0-2 | Monitoring tool not configured or data not pulled for the period; no coverage data available |

### Share of Voice Accuracy

| Score | Evidence |
|-------|----------|
| 9-10 | SOV weighted by outlet authority, estimated reach, and article prominence; trend line shows 3+ periods; methodology documented |
| 7-8 | SOV weighted by outlet tier; reach estimates included; 2+ periods compared; minor weighting inconsistencies |
| 5-6 | SOV based on mention count with outlet tier noted but not weighted; single-period snapshot only |
| 3-4 | Raw mention count used as SOV proxy; no weighting; competitor comparison incomplete |
| 0-2 | No SOV calculation performed; only own-brand mentions listed |

### Sentiment Classification

| Score | Evidence |
|-------|----------|
| 9-10 | Positive/neutral/negative classification with supporting evidence for each; dominant themes identified with article counts; negative trends flagged with urgency level |
| 7-8 | Sentiment classified for 90%+ of coverage; themes identified; some edge cases lack clear classification rationale |
| 5-6 | Sentiment classified for 70-90% of coverage; themes are generic ("product", "company"); negative items flagged without urgency context |
| 3-4 | Sentiment applied inconsistently; no thematic analysis; negative coverage not distinguished from neutral |
| 0-2 | No sentiment analysis performed; coverage listed without any qualitative assessment |

### Actionability of Flags

| Score | Evidence |
|-------|----------|
| 9-10 | Every flagged item includes outlet name, article summary, sentiment, potential impact assessment, and recommended response action with owner |
| 7-8 | Flagged items include context and recommended action; some items lack owner assignment |
| 5-6 | Items flagged but recommendations are generic ("monitor situation"); no impact assessment |
| 3-4 | Negative items listed but not prioritised; no recommended actions |
| 0-2 | No items flagged; report is purely descriptive with no alerts |

### Reporting Cadence

| Score | Evidence |
|-------|----------|
| 9-10 | Report delivered on schedule for 4+ consecutive periods; methodology consistent; format standardised |
| 7-8 | Report delivered on time for 3+ periods; minor methodology changes documented |
| 5-6 | Report delivered but sometimes 1-2 days late; methodology changes between periods without documentation |
| 3-4 | Reports delivered sporadically; inconsistent methodology makes period comparison unreliable |
| 0-2 | No regular reporting cadence established; reports produced ad hoc only |
