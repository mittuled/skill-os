# Framework: in-app-announcement-writer

Writes in-app announcement copy for feature releases optimised for discovery and activation.

## Framework: Benefit-Led Microcopy

### Approach

This skill uses the **Benefit-Led Microcopy** framework to structure the output.

### Sections

1. **Feature Context**: Define the content, evidence requirements, and format for this section.
2. **Headline and Body Copy (<40 words)**: Define the content, evidence requirements, and format for this section.
3. **CTA and Destination**: Define the content, evidence requirements, and format for this section.
4. **Publish and Expiration Rules**: Define the content, evidence requirements, and format for this section.

### Quality Checklist

- [ ] Feature Context completed with specific, evidence-based content
- [ ] Headline and Body Copy (<40 words) completed with specific, evidence-based content
- [ ] CTA and Destination completed with specific, evidence-based content
- [ ] Publish and Expiration Rules completed with specific, evidence-based content
- [ ] All assumptions stated explicitly with source
- [ ] Recommendations are specific, actionable, and prioritised
- [ ] Document reviewed by at least one stakeholder
