# Tool Categories Reference

Standard SaaS categories for company tooling onboarding. MCP availability status is approximate and should be verified against the latest MCP server registry at onboarding time.

## Communication

| Tool | MCP Available | Notes |
|------|--------------|-------|
| Slack | Yes | Official MCP server available |
| Microsoft Teams | Partial | Community MCP server, limited API surface |
| Discord | Yes | Community MCP server |
| Zoom | No | REST API only |
| Google Chat | No | REST API via Google Workspace |
