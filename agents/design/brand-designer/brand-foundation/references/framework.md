# Framework: Brand Foundation

Defines the structural components and research methods for establishing a brand's foundational pillars before visual identity work begins.

## Foundation Architecture

### Core Components

| Component | Definition | Output Format | Required |
|-----------|-----------|---------------|---------|
| Mission | Why the company exists (beyond profit) | Single sentence | Yes |
| Vision | Future state the company is building toward | Single sentence | Yes |
| Values | 3-5 guiding principles that govern decisions | Name + one-sentence definition + behavioural example | Yes |
| Personality | Brand character traits on a spectrum | 3-5 trait spectra (e.g., "confident, not arrogant") | Yes |
| Tone of Voice | How the brand communicates | Attribute + do/don't copy examples per context | Yes |
| Visual Principles | 3-5 design directives derived from values | Principle name + one-sentence directive + rationale | Yes |

### Personality Spectrum Model

For each personality trait, define two poles and the brand's intended position:

```
← More formal ——————————●——————— More casual →
← More serious ————●————————————— More playful →
← More sophisticated ——————●——————— More accessible →
← More authoritative ●——————————————— More collaborative →
```

Position must be deliberate and traceable to values.

## Stakeholder Discovery Methods

### Workshop Techniques

| Technique | Purpose | Output |
|-----------|---------|--------|
| Brand archetype sorting | Surface personality preferences | Archetype ranking with rationale |
| Anti-brand exercise | Define what the brand is NOT | Exclusion list |
| Competitor landscape mapping | Identify whitespace | Positioning matrix |
| Future headline | Articulate ambition | Vision statement candidates |
| Brand mantra (3-word filter) | Distil essence | Candidate mantras |

### Competitive Positioning Matrix

Plot competitors on two axes most relevant to the category (e.g., formal/casual vs. technical/accessible). The matrix reveals:
- Crowded quadrants (avoid or differentiate sharply)
- Whitespace positions (potential differentiation opportunity)
- Category conventions (to follow or subvert deliberately)

## Brand Values Quality Standards

A well-formed brand value must:
1. Be specific enough to guide a decision (not generic like "integrity" without context)
2. Have a behavioural definition (what does someone do when living this value?)
3. Create a potential trade-off (if a value never conflicts with anything, it has no discriminating power)
4. Be memorable: 3-7 words for the name/label

## Tone of Voice Context Matrix

| Context | Emotional Register | Example Purpose |
|---------|------------------|----------------|
| Onboarding | Encouraging, clear | Build confidence in a new user |
| Error state | Empathetic, direct | Reduce frustration, enable recovery |
| Success confirmation | Celebratory, brief | Reinforce positive behaviour |
| Destructive action | Serious, unambiguous | Prevent accidental data loss |
| Marketing copy | Expressive, aspirational | Attract and persuade |
| Legal/compliance | Formal, precise | Ensure comprehension |
