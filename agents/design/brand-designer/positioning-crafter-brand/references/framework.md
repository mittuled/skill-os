# Framework: Positioning Crafter Brand

Defines the methodology for translating positioning strategy into concrete visual design language.

## Positioning-to-Design Translation System

### Positioning Canvas Dimensions

Before mapping to visual design, extract the positioning on three axes (Jobs/Pains/Gains from Osterwalder):

| Dimension | Research Question | Design Implication |
|-----------|------------------|-------------------|
| Functional Job | What task does the product help users accomplish? | Information density, efficiency signals in layout |
| Emotional Job | How does the product make users feel? | Colour temperature, motion personality, image style |
| Social Job | How does the product affect users' self-image? | Premium vs. accessible cues, peer signals |
| Pain Relieved | What friction does it eliminate? | Simplicity, clarity, removal of visual noise |
| Gain Created | What new value does it create? | Aspiration, expansion, openness in visual language |

### Positioning-to-Visual-Design Mapping Table

| Positioning Attribute | Colour | Typography | Layout | Imagery | Motion |
|----------------------|--------|-----------|--------|---------|--------|
| Premium / Enterprise | Cool neutrals, restrained accent | High-contrast serifs or geometric sans | Generous whitespace, low density | Professional photography, minimal illustration | Slow, deliberate transitions |
| Accessible / Friendly | Warm tones, approachable accent | Rounded sans-serif, medium weight | Moderate density, ample padding | Inclusive photography, friendly illustration | Bouncy, energetic micro-interactions |
| Technical / Expert | Dark backgrounds optional, precise accent | Monospace elements, tight tracking | High density, information-forward | Diagrams, code, data visualisations | Functional feedback, minimal decoration |
| Innovative / Bold | High contrast, unexpected accent | Variable or expressive display type | Asymmetric, rule-breaking layout | Abstract, conceptual visuals | Fast, kinetic, attention-grabbing |
| Trustworthy / Stable | Conservative palette, muted accents | Legible, proven typefaces | Structured grid, predictable patterns | Real people, documentary photography | Minimal; animation only when purposeful |

## Visual Direction Exploration Protocol

For each positioning direction board, document:
1. **Rationale**: Which positioning attributes drive each design choice
2. **Touchpoints shown**: Must include at least one digital (web/app) and one print/OOH application
3. **Intensity spectrum**: Show the direction at 60%, 80%, and 100% expression intensity
4. **Conflict check**: Note any conflict with the existing brand identity system

## Direction Evaluation Criteria

| Criterion | Weight | Evaluator |
|-----------|--------|---------|
| Positioning fidelity | 35% | Brand strategist / product marketing |
| Identity system compatibility | 30% | Brand designer |
| Audience resonance | 20% | User research / target audience review |
| Distinctiveness from competitors | 15% | Competitive audit |

## Application Example Requirements

Each application example must show the positioning direction applied to a real asset:
- One digital UI touchpoint (web hero, app screen)
- One social media format
- One print or OOH format (if applicable)
- Annotations linking each visual choice back to the positioning attribute it expresses
