# Framework: Brand Identity V1

Defines the structural components and design system hierarchy for a complete v1 visual identity.

## Identity System Architecture

### Component Hierarchy

```
Brand Identity System
├── Core Marks
│   ├── Primary logo (full lockup)
│   ├── Secondary logo (compact)
│   ├── Logomark (icon only)
│   └── Wordmark (text only)
├── Colour System
│   ├── Primary palette (1-2 colours)
│   ├── Secondary palette (2-3 colours)
│   ├── Neutral palette (3-5 greys + white/black)
│   └── Semantic colours (success, warning, error, info)
├── Typography System
│   ├── Display typeface (headings H1-H2)
│   ├── Body typeface (H3-H6, body, UI)
│   └── Monospace (code, data — if needed)
├── Supporting Elements
│   ├── Iconography style
│   ├── Illustration direction
│   ├── Photography guidelines
│   └── Layout & spacing principles
└── Usage Guidelines
    ├── Clear space rules
    ├── Minimum sizes
    ├── Incorrect usage examples
    └── File format specifications
```

## Design Quality Gates

| Element | Gate | Standard |
|---------|------|---------|
| Logo | Minimum size | Legible at 16px (favicon) and 32px (app icon) |
| Colour | Accessibility | All text combinations pass WCAG 2.1 AA (4.5:1 body, 3:1 large) |
| Colour | Distinctiveness | Primary colour must score <70% similarity to top 3 competitors |
| Typography | Web performance | Total font weight < 200KB for primary typeface |
| Typography | Hierarchy clarity | H1 must be 2.5× body size minimum |
| Logo | Versatility | Must work on white, black, and primary brand colour backgrounds |

## Brand Architecture Models

| Model | Structure | When to Use |
|-------|-----------|-------------|
| Monolithic | Single master brand; all products share the same name and identity | Startup with one product; strong brand equity goal |
| Endorsed | Sub-brands exist with parent brand visible (e.g., "X by Acme") | Portfolio with distinct audiences; credibility transfer needed |
| House of Brands | Separate brands, parent company invisible (e.g., P&G model) | Distinct audiences that would conflict under one brand |

## Interbrand Brand Equity Dimensions (Reference)

Design decisions should support long-term equity across:
- **Clarity**: Is the brand's role and purpose clear?
- **Commitment**: Does leadership invest in the brand?
- **Governance**: Are usage rules enforced?
- **Responsiveness**: Can the identity adapt to new contexts?
- **Authenticity**: Does the identity reflect genuine organisational character?
- **Differentiation**: Does the identity stand apart from competitors?
- **Consistency**: Is the identity applied uniformly across touchpoints?
- **Presence**: Does the brand occupy a prominent place in users' minds?
- **Engagement**: Does the identity resonate emotionally with audiences?
- **Performance**: Does the brand deliver on its promise at every touchpoint?
