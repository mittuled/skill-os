# Checklist: Magic Quadrant Strategy

Submission readiness and year-round evaluation management checklists.

## Checklist 1: Submission Readiness

Complete before submitting any formal evaluation questionnaire.

### Strategy and Positioning
- [ ] Target quadrant / tier position identified and documented
- [ ] Gap analysis completed: current position vs. target position on each scored criterion
- [ ] Positioning narrative drafted: connects company vision, product capabilities, market traction, and customer outcomes to evaluation criteria
- [ ] Narrative reviewed and approved by CEO and CPO
- [ ] All positioning claims cross-checked against public statements (press releases, website, investor materials)
- [ ] No claimed capabilities that are on the roadmap but not yet available, unless clearly presented as roadmap items

### Content Completeness
- [ ] All required questionnaire sections completed — no blanks or "TBD"
- [ ] Product capability responses aligned with current product documentation
- [ ] Financial / business metrics approved by CFO or Finance lead
- [ ] Customer count, revenue range, and growth rate confirmed with Finance
- [ ] Geographic presence data verified
- [ ] Roadmap section reviewed and limited to items the company can commit to publicly

### Customer References
- [ ] Minimum number of references met per evaluation requirements (typically 5–15 for Gartner MQ)
- [ ] Reference tier mix reviewed: minimum 50% Tier A references
- [ ] All listed references are current customers (not churned)
- [ ] All references have been briefed on evaluation context, analyst questions, and company key messages
- [ ] Reference availability confirmed for the evaluation window
- [ ] No reference used >3 times in the last 18 months
- [ ] CS health scores checked for all references — no detractors on the reference list

### Internal Consistency
- [ ] Submission reviewed against the previous year's submission for contradictions
- [ ] Submission reviewed against all currently active evaluation submissions at other firms
- [ ] Product claims verified by Product Lead
- [ ] Customer experience claims verified by Customer Success Lead
- [ ] Sales execution claims (win rates, deal counts, geographic coverage) verified by Sales
- [ ] Legal review completed for any claims involving regulatory compliance or certifications

### Demo Preparation (if demo required)
- [ ] Demo environment confirmed as stable and current
- [ ] Demo flow aligned to evaluation criteria being scored
- [ ] Demo rehearsed with all participants
- [ ] Roles assigned: who presents each section, who handles live Q&A
- [ ] Backup demo environment available
- [ ] Technical prerequisites confirmed with analyst firm

## Checklist 2: Year-Round Evaluation Management

Complete quarterly to maintain evaluation readiness.

### Q1 (Post-Evaluation Debrief)
- [ ] Post-publication analysis completed: strengths vs. target, cautions vs. gap-closure plan
- [ ] Analyst debrief call scheduled (within 60 days of publication) to get detailed feedback
- [ ] Gap-closure plan actioned: product gaps assigned to roadmap, perception gaps assigned to briefing plan
- [ ] Reference customer debrief completed: thank you sent, availability noted for next cycle
- [ ] AR tracker updated with all evaluation-related commitments and outcomes

### Q2 (Narrative Development)
- [ ] Annual strategy briefing scheduled with primary evaluator(s)
- [ ] Customer reference pool audited: add new Tier A references, remove fatigued or at-risk references
- [ ] Product proof points from Q1 releases captured and formatted for analyst use
- [ ] Competitive intelligence reviewed: has any competitor's position changed in analyst coverage?
- [ ] Positioning narrative updated to reflect product and market changes

### Q3 (Execution Evidence)
- [ ] Product briefing scheduled with evaluation analyst
- [ ] New customer wins and retention data captured for evaluation evidence
- [ ] Any cautions from previous evaluation assessed: have they been addressed? Is evidence available?
- [ ] Evaluation timeline confirmed: when does the next cycle open?
- [ ] Reference pipeline confirmed: enough Tier A references available for next submission

### Q4 (Evaluation Preparation)
- [ ] Evaluation submission strategy document drafted
- [ ] Criteria mapping table updated with latest capabilities and evidence
- [ ] Internal kick-off meeting held: all contributors to the submission identified and aligned
- [ ] Submission timeline mapped to evaluation deadline with 2-week buffer
- [ ] All reference customers re-briefed for the upcoming cycle
- [ ] Demo environment built and tested (if demo required)
- [ ] Submission review process scheduled: product, legal, finance, CEO approvals calendared

---

## Checklist 3: Post-Publication Actions

Complete within 30 days of evaluation publication.

### Intelligence Extraction
- [ ] Report fully reviewed against analyst-report-monitor framework
- [ ] All strengths and cautions documented with exact analyst language
- [ ] Competitor positioning changes documented in competitive positioning tracker
- [ ] Internal intelligence brief completed and distributed

### Communication
- [ ] Analyst-cited strengths cleared for use in sales and marketing materials
- [ ] Sales team briefed on report findings: talking points prepared for buyer objections on cited cautions
- [ ] If Leaders placement: PR plan activated for press announcement
- [ ] If below target placement: internal stakeholder communication prepared (factual, forward-looking, not defensive)

### Gap Closure
- [ ] Each cited caution assigned an owner and response plan
- [ ] Product gaps escalated to product roadmap review
- [ ] Perception gaps (inaccurate cautions) assigned a briefing correction plan
- [ ] Gap-closure plan reviewed and approved by marketing leadership
- [ ] Next cycle target position confirmed based on gap-closure plan
