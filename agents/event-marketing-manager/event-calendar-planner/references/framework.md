# Framework: event-calendar-planner

This framework defines the annual event calendar cadence model, event type mix guidelines, budget allocation methodology, and conflict resolution rules for building a pipeline-optimised event programme.

## Event Type Classification

| Event Type | Format | ICP Funnel Stage | Typical Cost Range | Pipeline Contribution Model |
|------------|--------|-----------------|-------------------|-----------------------------|
| Tier 1: Flagship conference presence | In-person, sponsored or exhibiting | Mid/Late funnel | $20K–$150K+ | Direct pipeline; high-intent leads |
| Tier 2: Hosted field event | In-person, company-owned (dinner, roadshow, roundtable) | Mid funnel | $5K–$30K | Direct pipeline; targeted accounts |
| Tier 3: Webinar / virtual event | Virtual, company-owned | Top/Mid funnel | $1K–$10K | MQL generation; nurture acceleration |
| Tier 4: Partner co-marketing event | Virtual or in-person, co-hosted | Top funnel | $2K–$20K | Shared pipeline; new segment reach |
| Tier 5: User conference | In-person, company-owned flagship | Mid/Late funnel (retention + expansion) | $100K–$500K+ | Retention + expansion pipeline |
| Tier 6: Internal event (SKO, offsite) | In-person, internal | Internal | $50K–$300K+ | Indirect (sales enablement) |

## Annual Calendar Cadence Model

**Principle**: Distribute pipeline-generating events evenly across quarters to avoid Q4 clustering and Q1 dry spells.

| Quarter | Recommended Event Mix | Pipeline Contribution Goal |
|---------|----------------------|---------------------------|
| Q1 | 1–2 Tier 1 conferences + 1–2 webinars + 1 field event | 20–25% of annual event pipeline |
| Q2 | 1–2 Tier 1 conferences + 2–3 webinars + 1–2 field events | 25–30% of annual event pipeline |
| Q3 | 1 Tier 1 conference + 2–3 webinars + 1–2 field events + User conference (if applicable) | 25–30% of annual event pipeline |
| Q4 | 1 Tier 1 conference + 1–2 webinars + SKO planning (internal) | 15–20% of annual event pipeline |

**Warning signals requiring calendar rebalancing:**
- > 3 Tier 1 conferences in a single quarter: team capacity and lead processing risk
- 0 pipeline-generating events in any 6-week window: demand gen gap
- > 50% of annual events budget committed to a single event: concentration risk

## Budget Allocation Guidelines

| Event Tier | Recommended Budget Share | Rationale |
|------------|-------------------------|-----------|
| Tier 1 (flagship conferences) | 35–45% | Highest direct pipeline; requires premium investment |
| Tier 2 (hosted field events) | 15–20% | High conversion rate; targeted account engagement |
| Tier 3 (webinars) | 5–10% | High volume, low cost; MQL factory |
| Tier 4 (partner co-marketing) | 5–10% | Shared cost; new audience reach |
| Tier 5 (user conference) | 15–25% | Long-term retention and expansion ROI |
| Tier 6 (internal) | 10–15% | Indirect pipeline via enablement |
| Contingency | 10% | Opportunistic additions; mid-year pivots |

## Event Scoring Matrix

Score each candidate event on 5 criteria (1–5 scale):

| Criterion | Weight | Scoring Guide |
|-----------|--------|--------------|
| ICP attendance quality | 30% | 5 = > 60% ICP match; 3 = 30–50%; 1 = < 15% |
| Historical pipeline ROI | 25% | 5 = ≥ 5× cost in pipeline; 3 = 2–4×; 1 = no data or < 1× |
| Strategic timing | 20% | 5 = ideal pre-pipeline quarter; 3 = neutral timing; 1 = conflicts with launches or quarter-close |
| Competitive intelligence value | 15% | 5 = major competitors present; strategic insight available; 3 = minor value; 1 = no competitive value |
| Cost efficiency | 10% | 5 = CPL < $150; 3 = CPL $200–400; 1 = CPL > $500 |

**Minimum score to include in calendar:** Composite ≥ 3.0/5
**Score to prioritise for premium investment:** Composite ≥ 4.0/5

## Conflict Management with Product Calendar

| Conflict Type | Resolution Approach |
|--------------|--------------------| 
| Event overlaps with major product launch | Delay event sponsorship commitment by 2 weeks to confirm launch timeline; if overlap confirmed, use event as launch amplification or skip |
| Event falls in quarter-close week | Move company participation dates to avoid Q-close week; or send skeleton team only |
| Event competes for the same ICP as an owned event | Stagger by ≥ 3 weeks; consolidate messaging to avoid ICP fatigue |
| Budget conflict: two Tier 1 events in same month | Apply scoring matrix; commit to higher-scoring event; evaluate lower-scoring for attend-only |

## Quarterly Calendar Review Process

| Step | Activity | Output |
|------|----------|--------|
| 1 | Pull pipeline attribution per event for completed quarter | Attribution report from CRM |
| 2 | Compare actual CPL and pipeline per event vs. plan | Variance analysis |
| 3 | Identify bottom-quartile events (< 2× ROI) | Pause or cancel list |
| 4 | Identify emerging opportunities in remaining quarters | Opportunistic additions from contingency budget |
| 5 | Rebalance remaining budget: shift from underperformers to higher-ROI events | Updated budget allocation |
| 6 | Present revised calendar to marketing leadership for approval | Approved Q+1 calendar |
