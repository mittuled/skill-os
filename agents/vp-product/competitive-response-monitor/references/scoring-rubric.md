# Scoring Rubric: competitive-response-monitor

Evaluates the quality of a competitive signal assessment including signal capture, impact analysis, severity classification, and response recommendation.

## Criteria

| # | Criterion | Weight | Description |
|---|-----------|--------|-------------|
| 1 | Signal Capture | 20% | Specificity of the competitive move documentation with source, date, and confidence level |
| 2 | Customer Impact Assessment | 25% | Depth of evaluation across ICP segments with exposure ratings |
| 3 | Severity Classification | 25% | Accuracy of threat rating on differentiation impact and urgency axes |
| 4 | Response Recommendation | 30% | Quality of response options with effort, timeline, and trade-off analysis |

| **Total** | | **100%** | |

## Scale

Each criterion is scored **0-10**:
- **0**: No evidence / completely absent
- **5**: Partially present with significant gaps
- **10**: Fully present, comprehensive, no gaps

**Composite Score** = Σ (criterion score × weight)

## Grade Bands

| Grade | Composite Score | Label | Description | Recommended Action |
|-------|----------------|-------|-------------|-------------------|
| A+ | 9.0 – 10.0 | Exceptional | Signal precisely captured, segment-level impact assessed, severity correctly classified with evidence, response options fully evaluated with trade-offs | Execute recommended response |
| A | 8.0 – 8.9 | Strong | Signal well-documented, impact assessed across key segments, classification justified, response viable | Proceed with response after minor validation |
| B | 7.0 – 7.9 | Good | Signal captured, impact partially assessed, classification reasonable, response options outlined | Validate impact assessment before committing |
| C | 5.0 – 6.9 | Adequate | Signal captured but vague, impact guessed rather than assessed, classification present, response generic | Gather customer evidence before classifying |
| D | 3.0 – 4.9 | Weak | Signal anecdotal, no impact assessment, classification arbitrary, no response options | Restart with structured signal capture |
| F | 0.0 – 2.9 | Failing | No meaningful competitive monitoring performed | Establish competitive monitoring practice |

## Signal Tables

### Signal Capture

| Score | Evidence |
|-------|----------|
| 9-10 | Competitor move documented with specific feature/pricing/partnership details, primary source cited, date confirmed, confidence level rated (high/medium/low) |
| 7-8 | Move documented with specifics and source, confidence level noted |
| 5-6 | Move described in general terms, source identified but not primary |
| 3-4 | Move mentioned anecdotally without specifics or source |
| 0-2 | No competitive signal documented |

### Customer Impact Assessment

| Score | Evidence |
|-------|----------|
| 9-10 | Impact evaluated per ICP segment with exposure ratings, customer feedback on competitor capability collected, prospect evaluation patterns checked |
| 7-8 | Impact assessed across key segments with some customer evidence |
| 5-6 | Impact guessed based on feature overlap without customer input |
| 3-4 | Generic impact statement without segment-level analysis |
| 0-2 | No customer impact assessment |

### Severity Classification

| Score | Evidence |
|-------|----------|
| 9-10 | Threat rated on both differentiation impact and response urgency axes, classified as ignore/monitor/respond-at-pace/respond-urgently with detailed rationale |
| 7-8 | Both axes rated, classification applied with rationale |
| 5-6 | Classification applied but only one axis considered |
| 3-4 | Generic "high/low" classification without structured framework |
| 0-2 | No severity classification |

### Response Recommendation

| Score | Evidence |
|-------|----------|
| 9-10 | Multiple response options evaluated (accelerate roadmap, counter-feature, positioning adjustment, pricing change), each with effort estimate, timeline, and expected impact; trade-offs documented; clear recommendation with escalation path |
| 7-8 | Response options outlined with effort and timeline, recommendation made |
| 5-6 | Single response proposed without alternatives or trade-off analysis |
| 3-4 | Vague response suggestion ("we should do something") without specifics |
| 0-2 | No response recommendation |
