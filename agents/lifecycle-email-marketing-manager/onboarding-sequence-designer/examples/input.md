# Scenario: Post-Signup Onboarding for Project Management Tool

Design an onboarding email sequence for a project management SaaS tool targeting small team leads. The product has a freemium model with ~3,000 new signups per month.

## Product Context

- **Product**: TaskFlow — lightweight project management for teams of 3-15
- **Activation moment**: User creates their first project and invites at least one teammate (validated: users who do this within 5 days retain at 4x the rate of those who don't)
- **Current signup-to-activation rate**: 18%
- **Target**: 30% within 90 days of sequence launch
- **Time-to-activation median**: Currently 6.2 days for users who activate

## Milestones Between Signup and Activation

1. Account created (signup)
2. Profile completed (name, role, avatar)
3. First project created
4. First task added to project
5. Teammate invited (activation moment)

## Known Drop-off Points

- 40% of users never create their first project
- Of those who create a project, 25% never add a task
- Of those who add tasks, 35% never invite a teammate
