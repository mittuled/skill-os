# Entity Formation Checklist

Step-by-step checklist for forming a legal entity from document preparation through post-formation setup.

## Pre-Filing Preparation

- [ ] Entity type confirmed by General Counsel (C-Corp, LLC, S-Corp, etc.)
- [ ] State of incorporation selected with jurisdiction analysis on file
- [ ] Name availability checked with Secretary of State (online or by phone)
- [ ] Name reservation filed if formation will be delayed (typically 60-120 day hold)
- [ ] Registered agent selected and engaged in state of incorporation
- [ ] Registered agent engaged in each state where company will do business (foreign qualification)

## Certificate of Incorporation / Articles of Organization

### C-Corp (Delaware Certificate of Incorporation per DGCL 102)
- [ ] Company legal name (must include "Inc.", "Corp.", "Corporation", or "Company")
- [ ] Registered agent name and address in Delaware
- [ ] Purpose clause (typically "any lawful act or activity" per DGCL 102(a)(3))
- [ ] Authorized share structure:
  - [ ] Common Stock: [number] shares, par value $[0.00001-0.001]
  - [ ] Preferred Stock (blank check): [number] shares, par value $[0.00001-0.001]
  - [ ] Standard: 10,000,000 Common + 5,000,000 Preferred for seed-stage
- [ ] DGCL 102(b)(7) exculpation provision (director liability elimination for duty of care breaches)
- [ ] Incorporator name and signature

### LLC (Articles of Organization)
- [ ] Company legal name (must include "LLC" or "L.L.C.")
- [ ] Registered agent name and address
- [ ] Management structure: member-managed or manager-managed
- [ ] Purpose clause
- [ ] Operating Agreement drafted (not filed with state but legally required)

## State Filing

- [ ] Formation documents filed with Secretary of State
- [ ] Filing fee paid:
  - Delaware C-Corp: $89 (standard) or $189 (24-hour expedited)
  - Delaware LLC: $90
  - [Other state]: $[amount]
- [ ] Expedited processing requested if needed (24-hour, same-day, 2-hour available in Delaware at additional cost)
- [ ] Filing confirmation and entity number received
- [ ] Stamped formation documents saved to corporate minute book

## Federal Registrations

- [ ] EIN (Employer Identification Number) obtained from IRS (Form SS-4, online or by phone)
- [ ] EIN confirmation letter (CP-575) saved to corporate records
- [ ] S-Corp election filed if applicable (Form 2553 within 75 days of formation or by March 15)

## State Registrations

- [ ] State tax registration completed in state of incorporation
- [ ] State tax registration completed in each state where company has nexus (employees, office, significant sales)
- [ ] Foreign qualification filed in each state where company will "transact business":
  - [ ] Certificate of Good Standing obtained from state of incorporation
  - [ ] Application for Authority/Qualification filed with foreign state
  - [ ] Registered agent designated in foreign state
  - [ ] Filing fees paid
- [ ] State unemployment insurance registration completed in states with employees
- [ ] State workers' compensation registration/insurance obtained if required

## Corporate Records Assembly

### Corporate Minute Book Contents
- [ ] Stamped Certificate of Incorporation / Articles of Organization
- [ ] Bylaws (or Operating Agreement for LLC)
- [ ] EIN confirmation letter
- [ ] Registered agent agreements
- [ ] State tax registrations
- [ ] Foreign qualification certificates
- [ ] Initial board resolutions (once adopted)
- [ ] Stock ledger / membership interest ledger
- [ ] Blank stock certificates (if using certificated shares)

## Post-Formation Immediate Actions

- [ ] Bank account opened with EIN and formation documents
- [ ] Accounting system set up with correct entity type and fiscal year
- [ ] State annual report deadline noted and calendared
- [ ] Delaware franchise tax annual report deadline noted: March 1 (C-Corp) or June 1 (LLC)
- [ ] Entity information communicated to General Counsel for bylaws and board setup
- [ ] Handoff to bylaws-and-board-setup for governance establishment
- [ ] Handoff to founder-equity-issuance for share issuance

## Common Pitfalls

- [ ] Do NOT authorize too few shares (standard: 10M+ for C-Corp to allow option pool without amendment)
- [ ] Do NOT set par value too high (use $0.00001-$0.001 to minimize franchise tax under authorized shares method)
- [ ] Do NOT skip foreign qualification in states where company operates (penalties include inability to enforce contracts in state courts)
- [ ] Do NOT use personal SSN in place of EIN for corporate activities
- [ ] Do NOT operate or sign contracts before formation is complete (pre-incorporation liability risk to founders personally)
