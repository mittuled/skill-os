# Protection Checklists by Contract Type

This reference provides contract-type-specific protection checklists and universal protections applicable to all agreements.

## Universal Protections (All Contract Types)

These protections apply to every contract regardless of type. Check these first.

| # | Protection | Purpose |
|---|-----------|---------|
| U1 | Limitation of liability | Caps financial exposure to a defined amount |
| U2 | Indemnification | Allocates responsibility for third-party claims |
| U3 | IP ownership | Clarifies who owns intellectual property created or exchanged |
| U4 | Termination rights | Defines how and when either party can exit |
| U5 | Governing law | Specifies which jurisdiction's law applies |
| U6 | Dispute resolution | Defines process for resolving disagreements (arbitration, mediation, litigation) |
| U7 | Force majeure | Addresses performance obligations during extraordinary events |
| U8 | Data protection | Covers handling of personal data and compliance obligations |
| U9 | Confidentiality | Protects sensitive information shared between parties |
| U10 | Insurance requirements | Ensures adequate coverage for potential claims |

## SaaS Agreement (25 Items)

| # | Protection | Risk If Missing |
|---|-----------|----------------|
| S1 | Service level agreement (SLA) with uptime commitment | No recourse for downtime |
| S2 | SLA credits or remedies for breach | No financial consequence for poor performance |
| S3 | Data ownership and portability | Vendor lock-in, data loss on termination |
| S4 | Data backup and recovery obligations | Data loss without recourse |
| S5 | Security standards and certifications | Unknown security posture |
| S6 | Breach notification timeline | Delayed awareness of security incidents |
| S7 | Subprocessor disclosure and approval | Unknown third-party data access |
| S8 | Data deletion on termination | Data retained by vendor post-contract |
| S9 | API access and integration rights | Inability to integrate with other systems |
| S10 | Change notification for material changes | Surprise feature removal or terms changes |
| S11 | Subscription term and auto-renewal controls | Unintended contract extensions |
| S12 | Price escalation caps | Unpredictable cost increases |
| S13 | Audit rights | Inability to verify compliance claims |
| S14 | Escrow provisions for source code | Business continuity risk if vendor fails |
| S15 | Support response time commitments | No guaranteed support quality |
| S16 | Customization and configuration ownership | Loss of invested customization work |
| S17 | Export functionality | Inability to extract data in usable format |
| S18 | Compliance certifications (SOC 2, ISO 27001) | Unverified security practices |
| S19 | Liability cap proportional to contract value | Disproportionate risk allocation |
| S20 | Warranty on service conformance to documentation | No remedy for misrepresented features |
| S21 | Assignment and change of control | Contract transferred to unknown party |
| S22 | Survival clauses | Key obligations expire with contract |
| S23 | Entire agreement / integration clause | Side agreements create ambiguity |
| S24 | Notice provisions | Invalid communications |
| S25 | Amendment process | Unilateral contract changes |

| # | Protection | Risk If Missing |
|---|-----------|----------------|

| SV3 | Change order process | Uncontrolled scope changes |

| SV6 | Pre-existing IP carve-out | Disputes over background IP |
| SV7 | License to pre-existing IP used in deliverables | Dependency on vendor IP without rights |
| SV8 | Non-solicitation of employees | Talent poaching |
| SV9 | Representations and warranties | No recourse for misrepresentation |
| SV10 | Professional liability insurance | Uninsured errors and omissions |
| SV11 | Key personnel identification | Critical resources reassigned |
| SV12 | Subcontracting restrictions | Unknown parties performing work |
| SV13 | Background check requirements | Security risk from unvetted personnel |
| SV14 | Compliance with applicable laws | Regulatory exposure from vendor actions |
| SV15 | Audit rights over work product | Inability to verify quality |
| SV16 | Cure period for breach | Immediate termination without remedy opportunity |
| SV17 | Return of materials on termination | Company materials retained by vendor |
| SV18 | Conflict of interest disclosure | Hidden competing engagements |
| SV19 | Liability cap for professional services | Unlimited exposure from vendor errors |
| SV20 | Warranty period post-delivery | No recourse for latent defects |
| SV21 | Escalation procedures | No path to resolve disputes before litigation |
| SV22 | Status reporting requirements | No visibility into progress |

## NDA (12 Items)

| # | Protection | Risk If Missing |
|---|-----------|----------------|
| N1 | Definition of confidential information | Ambiguous scope of protection |
| N2 | Exclusions from confidentiality | Overly broad obligations |
| N3 | Permitted disclosures (legal compulsion) | No safe harbor for court orders |
| N4 | Non-use obligation | Information used for unintended purposes |
| N5 | Term of confidentiality obligation | Obligations expire too soon |
| N6 | Return or destruction of materials | Materials retained after NDA expires |
| N7 | Injunctive relief provision | Damages inadequate for breach |
| N8 | Residuals clause limitations | Counterparty claims right to use retained knowledge |
| N9 | No obligation to disclose | Compelled disclosure of sensitive information |
| N10 | No license grant | Implied IP rights from disclosure |
| N11 | Survival beyond termination | Obligations end with contract term |
| N12 | Marking requirements (if applicable) | Disputes over what constitutes confidential information |
