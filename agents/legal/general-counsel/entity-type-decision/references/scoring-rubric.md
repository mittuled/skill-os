# Scoring Rubric: entity-type-decision

Evaluates the quality and defensibility of an entity type and jurisdiction recommendation for a new or restructuring company.

## Criteria

| # | Criterion | Weight | Description |
|---|-----------|--------|-------------|
| 1 | Founder Intent Alignment | 25% | How well the recommendation matches founder goals (fundraising plan, exit strategy, tax preferences) |
| 2 | Tax Structure Optimality | 25% | Whether the entity type and jurisdiction minimize tax burden given the company's stage, revenue model, and founder residency |
| 3 | Fundraising Readiness | 20% | Whether the chosen structure supports the intended financing path (VC preferred stock, convertible notes, SAFE compatibility) |
| 4 | Jurisdiction Suitability | 15% | Quality of jurisdiction analysis covering franchise tax, case law predictability, investor familiarity, and privacy protections |
| 5 | Risk Disclosure Completeness | 15% | Whether material risks (conversion complexity, ongoing compliance, tax exposure) are identified and disclosed |
| **Total** | | **100%** | |

## Scale

Each criterion is scored **0-10**:
- **0**: No evidence / completely absent
- **5**: Partially present with significant gaps
- **10**: Fully present, comprehensive, no gaps

**Composite Score** = Σ (criterion score × weight)

## Grade Bands

| Grade | Composite Score | Label | Description | Recommended Action |
|-------|----------------|-------|-------------|-------------------|
| A+ | 9.0 – 10.0 | Exceptional | Recommendation fully aligns with founder intent, optimizes tax treatment, supports fundraising, jurisdiction analysis is comprehensive with cost estimates, and all material risks disclosed | Proceed to entity formation immediately |
| A | 8.0 – 8.9 | Strong | Recommendation is sound with minor gaps in one area (e.g., missing cost estimates for one jurisdiction) | Address minor gaps and proceed to formation |
| B | 7.0 – 7.9 | Good | Recommendation is reasonable but lacks depth in tax modeling or jurisdiction comparison | Supplement analysis before formation; acceptable for time-sensitive formations |
| C | 5.0 – 6.9 | Adequate | Recommendation covers basics but misses founder-specific factors or has superficial risk disclosure | Rework with additional founder input before proceeding |
| D | 3.0 – 4.9 | Weak | Recommendation defaults to standard choice without substantive analysis of alternatives | Reject and redo with proper comparative analysis |
| F | 0.0 – 2.9 | Failing | No meaningful analysis; recommendation is unsupported or contradicts stated founder goals | Escalate to General Counsel for complete rework |

## Signal Tables

### Founder Intent Alignment

| Score | Evidence |
|-------|----------|
| 9-10 | All founder inputs documented (fundraising timeline, exit strategy, residency, revenue model); recommendation explicitly maps to each input with reasoning |
| 7-8 | Most founder inputs captured; recommendation addresses primary goals but misses secondary considerations (e.g., personal tax implications) |
| 5-6 | Founder intent gathered informally; recommendation addresses fundraising plan but ignores tax preferences or exit strategy |
| 3-4 | Limited founder input collected; recommendation assumes VC-backed path without confirming |
| 0-2 | No founder intent gathering; entity type selected without consulting founders |

### Tax Structure Optimality

| Score | Evidence |
|-------|----------|
| 9-10 | Pass-through vs. double taxation modeled for founders' specific situations; state tax implications quantified; loss pass-through value calculated for early years |
| 7-8 | Tax comparison covers major entity types; founder-specific modeling present but incomplete (e.g., missing state-level analysis) |
| 5-6 | General tax comparison provided (C-corp vs. LLC); no founder-specific modeling |
| 3-4 | Tax mentioned but not analyzed; recommendation cites "standard practice" without tax reasoning |
| 0-2 | Tax implications not addressed |

### Fundraising Readiness

| Score | Evidence |
|-------|----------|
| 9-10 | Entity type confirmed compatible with target investor class; preferred stock issuance capability verified; SAFE/convertible note compatibility confirmed; investor-side tax concerns addressed |
| 7-8 | Entity supports intended financing instruments; minor gaps (e.g., S-corp shareholder limitations not discussed for potential investor types) |
| 5-6 | Entity type generally supports VC financing but no specific instrument compatibility analysis |
| 3-4 | Fundraising mentioned as a consideration but not analyzed against entity type constraints |
| 0-2 | Fundraising readiness not evaluated; entity type may block intended financing |

### Jurisdiction Suitability

| Score | Evidence |
|-------|----------|
| 9-10 | Comparison table covers 3+ jurisdictions with franchise tax estimates, case law assessment, investor familiarity rating, privacy provisions, and formation timeline |
| 7-8 | Two jurisdictions compared with cost estimates; case law and investor familiarity addressed |
| 5-6 | Primary jurisdiction selected with basic rationale; no formal comparison with alternatives |
| 3-4 | Jurisdiction mentioned but rationale is "industry standard" without supporting analysis |
| 0-2 | No jurisdiction analysis; default assumed without discussion |

### Risk Disclosure Completeness

| Score | Evidence |
|-------|----------|
| 9-10 | All material risks identified: conversion complexity, ongoing compliance burden, tax exposure, international implications; each risk has likelihood, impact, and mitigation guidance |
| 7-8 | Major risks identified with impact assessment; mitigation guidance present for most risks |
| 5-6 | Risks listed but without severity assessment or mitigation suggestions |
| 3-4 | Only obvious risks mentioned (e.g., "Delaware has franchise tax"); material risks like conversion complexity omitted |
| 0-2 | No risk disclosure or only boilerplate disclaimers |
