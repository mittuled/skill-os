# IP Assignment Checklist

Step-by-step verification checklist for ensuring complete intellectual property assignment to the company.

## Pre-Assignment Audit

- [ ] Identify all IP contributors: founders, employees, contractors, advisors, interns
- [ ] Determine each contributor's engagement start date and IP creation date range
- [ ] Classify contribution types: patentable inventions, copyrightable works (code, designs, content), trade secrets, trademarks
- [ ] Check for pre-existing IP that will be incorporated into company products
- [ ] Verify contributor residency/state for work-for-hire eligibility (California Labor Code 2870, Illinois Employee Patent Act, etc.)
- [ ] Confirm entity is legally formed and authorized to receive IP assignments

## Pre-Existing IP Assessment

- [ ] Each contributor discloses pre-existing IP on a schedule attached to the assignment agreement
- [ ] Pre-existing IP classified as: assign to company, license to company (exclusive/non-exclusive), or exclude
- [ ] Open-source contributions identified and license compatibility verified
- [ ] Prior employer obligations checked (non-compete, invention assignment carve-outs from previous employment)
- [ ] University or research institution IP policies reviewed for academic founders

## Agreement Requirements by Contributor Type

### Founders
- [ ] Present-tense assignment language ("hereby assigns") covering all IP created before and after incorporation
- [ ] Work-for-hire declaration for copyrightable works (where applicable under 17 USC 101)
- [ ] Moral rights waiver (to the extent permitted by jurisdiction)
- [ ] Invention disclosure obligation for future inventions during employment
- [ ] Consideration documented (equity issuance, employment, or nominal payment)
- [ ] Pre-existing IP schedule attached and signed

### Employees
- [ ] Standalone IP assignment agreement (not relying solely on employment agreement IP clause)
- [ ] State-specific carve-out language (CA 2870, IL 765 ILCS 1060, WA RCW 49.44.140, MN 181.78)
- [ ] Invention disclosure obligation with periodic reporting
- [ ] Work-for-hire declaration where copyright assignment is needed
- [ ] Clear scope definition: company-related work vs. personal projects

### Contractors
- [ ] Work-for-hire clause with fallback assignment provision (work-for-hire may not apply to all work categories under 17 USC 101)
- [ ] IP assignment covering all deliverables and work product
- [ ] Warranty of original work (no third-party IP infringement)
- [ ] No retained license or rights after assignment
- [ ] Subcontractor flow-down requirements for IP assignment

## Execution and Filing

- [ ] All agreements signed by both parties (wet signature or compliant e-signature)
- [ ] Execution log maintained with: contributor name, agreement type, execution date, witness/notary (if required)
- [ ] Patent assignment recordation filed with USPTO (if applicable, per 35 USC 261)
- [ ] Copyright assignment recordation filed with Copyright Office (if applicable, per 17 USC 205)
- [ ] Agreements stored in company document management system with restricted access
- [ ] Copies provided to each contributor

## Gap Remediation

- [ ] Unsigned agreements identified and escalation timeline set
- [ ] Ambiguous scope clauses flagged for supplemental assignment or confirmatory agreement
- [ ] State law limitations documented (overbroad clauses may be unenforceable)
- [ ] Former contributors who left without signing contacted for remediation
- [ ] Gaps documented in IP chain-of-title memo for due diligence readiness

## Due Diligence Readiness

- [ ] Complete IP contributor roster with current assignment status
- [ ] All executed agreements accessible in organized repository
- [ ] Pre-existing IP schedule current and complete
- [ ] Gap analysis report updated with remediation status
- [ ] IP chain of title traceable from original creator to company for all material IP
