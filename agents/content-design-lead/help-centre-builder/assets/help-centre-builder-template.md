[1-2 sentences: context and what the user will be able to do after following this guide]

## Content Style Guide (Help Centre)

[Voice and formatting rules specific to help content.]

| Rule | Detail |
|------|--------|
| Voice | [Same as product voice per content design spec; more explicit instruction language] |
| Person | Second person ("you"); never "the user" |
| Step verbs | Imperative: "Click", "Select", "Enter" — not "You can click" |
| Screenshots | Include at major decision points and non-obvious UI states |
| Avoid | "Simply", "just", "easy" — minimises user difficulty |
| Reading level target | Grade 8 or below (Flesch-Kincaid) |
| Title format | Question ("How do I...?") or task ("How to [action]") |
