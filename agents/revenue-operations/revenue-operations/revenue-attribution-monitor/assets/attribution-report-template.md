## 1. Attribution Model in Use

| Model | Description | When It Is Used |
|-------|-------------|----------------|
| **First-Touch** | 100% credit to the first touchpoint | Top-of-funnel analysis; brand awareness measurement |
| **Last-Touch** | 100% credit to the final touchpoint before close | Sales-close attribution; bottom-funnel analysis |
| **Linear** | Equal credit across all touchpoints | Balanced analysis across the full buyer journey |
| **Time-Decay** | More credit to touchpoints closer to close | Measuring which touchpoints accelerate deals |
| **Custom** | Defined weights per touchpoint type | Agreed model between marketing and sales |
