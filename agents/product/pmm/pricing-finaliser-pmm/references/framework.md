# Framework: Pricing Finalisation

Reference framework for locking final price points, building pricing page copy, and preparing sales reference materials.

## Price Point Finalisation Process

### Input Requirements
Before finalising, confirm these inputs are complete:

| Input | Source | Status Check |
|-------|--------|-------------|
| Pricing strategy document | PMM pricing-strategy-pmm | Approved by leadership |
| Packaging framework | PMM packaging-designer-pmm | Tier names, features, and upgrade triggers finalised |
| WTP data | Pricing research | Van Westendorp or Gabor-Granger results available |
| Competitive benchmarks | Competitor mapper | Current competitor price points documented |
| Margin targets | Finance | Unit economics model approved |
| Discount policy | Sales leadership | Discount bands and escalation rules agreed |

### Psychological Pricing Principles

| Principle | Application | Example |
|-----------|------------|---------|
| Charm pricing | End prices in 9 for consumer-adjacent products | $49/mo instead of $50/mo |
| Round pricing | Use round numbers for enterprise/premium positioning | $500/mo signals quality |
| Anchoring | Show the highest tier first to anchor perceived value | Enterprise tier displayed before Starter |
| Decoy effect | Structure middle tier to make the target tier look like the best deal | Pro tier priced to make Business tier the obvious choice |
| Annual discount | Offer 15-20% discount for annual commitment | "Save 20% with annual billing" |

## Pricing Page Copy Template

For each tier:

| Element | Guideline |
|---------|-----------|
| Tier name | One word, buyer-intuitive (Starter, Professional, Enterprise) |
| One-line description | Who this tier is for, not what it includes ("For growing teams") |
| Price display | Monthly price shown; annual price with savings callout |
| Feature list | 5-8 features per tier; highlight differentiating features, not every feature |
| CTA | Distinct per tier: "Start free", "Start trial", "Contact sales" |
| Recommended badge | Highlight the target tier with "Most popular" or "Best value" |

## Internal Pricing Reference Sheet

For sales team use:

| Section | Content |
|---------|---------|
| List prices | Published price per tier with billing frequency options |
| Discount bands | Standard: 0-10%, Manager: 10-20%, VP: 20-30%, Deal desk: 30%+ |
| Discount triggers | Multi-year commitment, strategic logo, competitive displacement |
| Escalation rules | When to involve deal desk (discount threshold, non-standard terms, custom tier) |
| Competitive intelligence | Competitor price points for the top 3 competitors with positioning notes |
| Objection handlers | "Too expensive" → reframe on value per outcome; "Competitor is cheaper" → differentiation response |
| Grandfathering policy | How existing customers are migrated; communication timeline |

## Pricing Lock Checklist

- [ ] Price points set for all tiers (within strategic price bands)
- [ ] Add-on pricing defined (if applicable)
- [ ] Annual vs. monthly pricing with discount percentage confirmed
- [ ] Finance sign-off on margin targets at each tier
- [ ] Sales leadership sign-off on pricing presentation and discount policy
- [ ] Pricing page copy written and reviewed
- [ ] Internal pricing reference sheet distributed to sales
- [ ] Enablement brief updated with pricing details
- [ ] CRM pricing fields updated
- [ ] Legal review of pricing terms (if pricing model changed)
