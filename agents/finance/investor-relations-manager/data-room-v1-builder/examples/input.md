# Data Room V1 Builder — Example Input

## Scenario

Meridian AI is raising its seed round and has received interest from three investors who want to begin diligence. The team needs to set up their first data room. They have some documents ready but are missing others. The IR Manager needs to inventory what's available and identify gaps before sharing access.

## Input JSON

```json
{
  "company_name": "Meridian AI",
  "fundraising_stage": "seed",
  "documents_available": [
    "Pitch deck",
    "Executive summary",
    "Financial model",
    "Historical P&L",
    "Bank statements",
    "Certificate of incorporation",
    "Cap table",
    "Founder agreements",
    "IP assignment agreements",
    "Product roadmap",
    "Demo video",
    "Team bios",
    "Org chart",
    "ARR history",
    "Key metrics dashboard"
  ]
}
```
