# SAFE Note Setup — Example Input

## Scenario

Meridian AI is raising a $750K pre-seed via YC Post-Money SAFEs. Marcus Reed, an angel investor, wants to invest $125K. The company is offering a $8M valuation cap with a 20% discount. The IR Manager needs to prepare the SAFE terms, model conversion scenarios for the investor, and generate the execution checklist.

## Input JSON

```json
{
  "company_name": "Meridian AI",
  "investor_name": "Marcus Reed",
  "date": "2026-01-15",
  "safe_type": "post_money",
  "principal_usd": 125000,
  "valuation_cap_usd": 8000000,
  "discount_rate": 0.20,
  "pro_rata_rights": true
}
```
